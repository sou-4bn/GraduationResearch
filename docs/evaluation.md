# アクセス評価指標

## 目的
観光地へのアクセスを「行けるか」ではなく「現実的に行きやすいか」で評価する。

## 評価式
C = T_invehicle + 2.0 × T_wait + 1.5 × T_walk + 10 × N_transfer + 0.5 × max(0, W_max - 10)

## 変数
- T_invehicle: 乗車時間合計（分）
- T_wait: 待ち時間合計（分）
- T_walk: 徒歩時間合計（分）
- N_transfer: 乗り換え回数
- W_max: 最長待ち時間（分）

## 補足
- 待ち時間は乗車時間より負担が大きいとみなす
- 徒歩時間も追加負担として評価する
- 長すぎる待ち時間には追加ペナルティを与える

## 評価区分
- ◎ : C ≤ 60
- △ : 60 < C ≤ 100
- × : C > 100
