# 地方観光アクセスマップ（複数ファイル版）

## 構成
- index.html: 画面本体
- css/style.css: UIデザイン
- js/app.js: 起動処理
- js/map.js: 地図表示とマーカー制御
- js/ui.js: 検索・詳細シートのUI制御
- js/utils.js: 距離計算と評価ロジック
- js/data.js: CSVから生成したデータ
- data/*.csv: 元データ

## ポイント
- 最初から観光地の一覧は出さず、地図上のマーカー選択を中心にしたUIです。
- 見た目は最新アプリ風に、上部サーチ・フィルタチップ・下部ボトムシートで構成しています。
- 現状の評価は最寄り交通機関までの距離ベースです。

## 開き方
`index.html` をブラウザで開くだけで動きます。
インターネット接続があれば、地図タイルとLeaflet CDNが読み込まれます。
